## 当前应用

### 技术栈

- **框架**: Next.js 14
- **语言**: React 18 + TypeScript
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **后端**: Firebase (Auth + Firestore)
- **动画**: Framer Motion
- **部署**: Vercel / EdgeOne Pages

### 主要特性

- ✅ 三种车型配置 (Road/MTB/Fold)
- ✅ 完整的组件选择系统
- ✅ 实时价格和重量计算
- ✅ 配置保存和管理
- ✅ 深色/浅色主题切换
- ✅ 国际化 (中英文)
- ✅ Firebase集成
- ✅ 响应式设计
- ✅ 完整的测试覆盖
