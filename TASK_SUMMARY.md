### 2. 创建高保真原型

#### 原型文件: `prototype.html`

**主要特性**:

1. **三车型完整支持**
   - 公路车 (Road): 专业竞速配置，轻量化设计
   - 山地车 (MTB): 强劲避震系统，崎岖山路适配
   - 折叠车 (Fold): 便携折叠设计，城市通勤首选

2. **响应式设计**
   - 桌面端 (≥1024px): 三列布局，完整功能展示
   - 平板端 (768-1023px): 双列布局，侧边栏优化
   - 移动端 (<768px): 单列布局，底部固定操作栏

3. **深色主题设计系统**
   - 主色调: 蓝色系 (#3b82f6)
   - 强调色: 橙色系 (#f97316)
   - 渐变网格背景 + 噪点纹理叠加
   - 毛玻璃效果模态框

4. **流畅动画交互**
   - 模态框缩放进入动画
   - 价格数字滚动更新
   - 卡片悬停上浮效果
   - Toast 通知滑入动画

5. **完整功能实现**
   - 车型切换（带组件数据联动）
   - 组件选择模态框（按类别筛选）
   - 实时价格和重量计算
   - 配置保存/分享/重置
   - 键盘快捷键支持（ESC 关闭模态框）

**技术实现**:
- HTML5 + CSS3 + Vanilla JavaScript
- Tailwind CSS CDN
- Google Fonts (Space Grotesk + Inter)
- Lucide Icons (SVG inline)
